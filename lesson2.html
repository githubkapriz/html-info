Теги для розмітки сторінки

  1. <header>…</header> − визначає шапку сайту.
  2. <nav>…</nav> − призначений для створення навігаційних меню.
  3. <aside>…</aside> − призначений для створення бокових панелей (сайдбарів).
  4. <article>…</article> − допомогає створювати нові матеріали на сторінці, додавати контент до сторінки.
  5. <section>…</section> − задає розділ документа, може застосовуватися для блоку новин, контактної інформації, глав тексту, вкладок в діалоговому вікні тощо; зазвичай містить заголовок. Допускається вкладати один тег
    всередину іншого.
  6. <footer>…</footer> − визначає нижню частину сторінки, тобто підвал.
